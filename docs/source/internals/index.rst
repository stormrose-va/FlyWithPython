Project Documentation
=====================

This part of the documentation is for people hacking on FlyWithPython itself.
It is a mandatory source of information for anyone wishing to contribute to
FlyWithPython's development. This guide describes code standards as well as
processes and tools around the management of FlyWithPython's source code.

.. important::

   By contributing to this project, the contributor accepts all contributions
   (code, documentation, etc.) are publicly made available and become subject
   to the terms of the `MIT License`_.
